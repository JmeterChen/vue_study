#### 2022-1-15  组件化开发学习3
- 父子组件的通信复杂案例
  - 实现思路及要点：
  1. 子组件接受父组件传入的数据
  2. 不能再子组件中单纯修改接受父组件数据的变量，不然vue会提示报错，建议在子组件中用新定义的变量承接进行修改其值；
  3. 子组件 input 框中的数据改变时 希望子组件中的数据和父组件传入变量均发生变化；
  4. 子组件1 input 框中的数据希望是 子组件2 input 框的数据 百分之一，这里注意两个input 框变化时都要触发数据更新；
